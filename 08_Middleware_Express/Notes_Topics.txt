બધું middleware થી થાય છે.

Middleware શું છે?

Middleware = Request અને Response ની વચ્ચે ચાલતું function

Flow:

Request → Middleware → Route → Response


Simple Meaning

User request આવે ત્યારે:

1. પહેલા middleware run થાય
2. પછી route run થાય
3. પછી response જાય


next() શું છે?

next() route તરફ control મોકલે છે.

જો next() નહીં લખો તો request અટકી જશે.

Logger Middleware

Logger requests track કરવા માટે use થાય છે.

Real-time:

Server monitoring
API tracking
Security logs

code:

const logger = (req,res,next)=>{
    console.log("Middleware Called");

    console.log(req.method);
    console.log(req.url)
    next();
}

// middleware call
app.use(logger);


app.get("/",(req,res)=>{
    res.send("Home page")
})

Body Parser Middleware

POST request માં data receive કરવા use થાય છે.

Express 4.16+ પછી:

Body Parser Middleware નો main કામ 👉
POST request માંથી આવતો data વાંચવો (receive કરવો).

Simple Meaning

Client / Postman / Frontend data મોકલે:

{
  "name":"shubham",
  "city":"ahmedabad"
}

આ data server સુધી આવે છે.
પણ Express by default JSON data સમજી શકતું નથી.

Wihtouit kraru undefined aavse
app.use(express.json());

આ middleware JSON data ને JavaScript object માં convert કરે છે.

app.use(express.json());

app.post("/users",(req,res)=>{
    console.log("url : ",req.url,"method : ",req.method)

    console.log("body :",req.body)

    res.send("Data Received");
})
