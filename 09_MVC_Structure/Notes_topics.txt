MVC backend development નું industry standard architecture છે.

Used in:

Netflix
Amazon
Flipkart

MVC શું છે?
MVC =

Letter	Meaning
M	    Model
V	    View
C	    Controller

Simple      Meaning
Part	    Work
Model	    Database handle કરે
View	    Frontend/UI
Controller	Logic handle કરે

Node.js backend માં mostly:

Routes
Controllers
Models

આ structure use થાય છે.

MVC કેમ Important?

Without MVC:

❌ બધું code app.js માં આવે
❌ Project messy બને
❌ Large project manage મુશ્કેલ

MVC પછી:

✅ Clean code
✅ Easy maintenance
✅ Team work easy
✅ Reusable code
✅ Professional structure

STEP 5 — Create Controller
controllers/userController.js
const getUsers = (req, res) => {

    res.send("All Users");

};

module.exports = {
    getUsers
};


Controller શું કરે?

Controller business logic handle કરે.

Examples:

User data
Product logic
Login logic
API response


STEP 6 — Create Routes
routes/userRoutes.js
const express = require("express");

const router = express.Router();

const userController = require("../controllers/userController");

router.get("/", userController.getUsers);

module.exports = router;

router શું છે?
express.Router()

Mini Express app જે routes manage કરે.
