REST API શું છે?

REST = Representational State Transfer

Simple ભાષામાં:

Frontend ↔ Backend communication system

Frontend request મોકલે
Backend response આપે

Mostly JSON format માં.

STEP 1 — Create Project

CMD:

mkdir rest-principles
cd rest-principles
STEP 2 — Initialize Node
npm init -y
STEP 3 — Install Express
npm install express
STEP 4 — Create app.js
app.js
const express = require("express");

const app = express();

app.use(express.json());

app.listen(3000, () => {

    console.log("Server Running");

});
express.json() કેમ use થાય?

POST/PUT request માં JSON data read કરવા.

STEP 5 — GET API
Get All Products
app.get("/products", (req, res) => {

    res.json({

        message: "All Products"

    });

});
URL
localhost:3000/products
Output
{
   "message":"All Products"
}
STEP 6 — POST API
Add Product
app.post("/products", (req, res) => {

    console.log(req.body);

    res.status(201).json({

        message: "Product Added"

    });

});
Postman JSON
{
   "name":"Laptop",
   "price":50000
}
Output
{
   "message":"Product Added"
}
STEP 7 — GET Single Product
app.get("/products/:id", (req, res) => {

    const id = req.params.id;

    res.json({

        message: "Single Product",
        productId: id

    });

});
URL
localhost:3000/products/10
Output
{
   "message":"Single Product",
   "productId":"10"
}
STEP 8 — UPDATE API
app.put("/products/:id", (req, res) => {

    const id = req.params.id;

    res.json({

        message: "Product Updated",
        productId: id

    });

});
STEP 9 — DELETE API
app.delete("/products/:id", (req, res) => {

    const id = req.params.id;

    res.json({

        message: "Product Deleted",
        productId: id

    });

});

Important Interview Questions
What is REST API?

REST API is an architecture style for communication between client and server using HTTP methods.

What is Resource in REST?

Resource is any data object accessible through URL.

Why JSON Used?

JSON is lightweight and easy data exchange format.
